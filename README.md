# 行业分析 Skill

> **v1.1** · 更新 2026-07-04 · License: MIT（见 `LICENSE`）
> 运行前提：需运行环境具备**联网检索能力**（WebSearch）。估值分位取数脚本依赖 `akshare`/`pandas` 为**可选增强**，不装也能用全部核心功能。

输入一个行业 / 细分主题 / 一家公司，产出一份"**看懂产业 + 找到投资机会**"的结构化研究报告（HTML 可视化交付物）。

---

## ⚠️ 定位：这是「研究框架工具」，不是「决策系统」

请先读懂这一点，再使用：

| 它是什么 | 它不是什么 |
|---|---|
| 把产业拆解、价值判断、估值赔率**结构化**的框架 | ❌ 不是能直接告诉你"买什么"的荐股系统 |
| 用**公开联网数据**填充、并交叉校验的研究助手 | ❌ 不接 Wind/Bloomberg，无实时行情、无一手调研 |
| 帮你**快速建立认知地图 + 提出该问的问题** | ❌ 不替代你的独立判断与尽职调查 |
| 输出**可解释的结构化观点**（带证伪条件、置信标注）| ❌ 不保证数字准确（可能滞后/有误），不是结论 |

**核心价值在"框架"，不在"答案"。** 它的真实能力边界等同于"一个有联网能力的强模型 + 一套被固化下来的分析方法论"——价值是**一致性、省力、合规、可视化**，而不是拿到了别人拿不到的数据或能力。

> 所有标的均为**产业逻辑映射**，非投资建议。报告默认附风险/证伪条件与免责声明。投资有风险，决策需结合个人风险承受能力并做独立尽调。

## 它能帮你做什么

1. **产业链图谱** —— 上游/中游/下游拆解，按价值量热力上色，一眼看清"钱在哪个环节"。
2. **高价值方向识别** —— 高增长/高利润/高壁垒"三高"打分 + 热力矩阵，**并挂联网财务数据交叉校验**（主观分必须有数据兜底，背离要么解释要么下修）。
3. **价值方向下探** —— 高价值环节继续下钻到细分赛道 / 技术路线 / 卡位点。
4. **产业标的梳理** —— 龙头 / 弹性 / 隐形冠军逻辑映射表。
5. **估值与赔率** —— PEG、历史估值分位、隐含预期差，"价值 × 估值"四象限。**好行业 ≠ 好价格**。（预测类数字取不到可靠来源时降级为定性区间，不编造。）
6. **投资机会综合** —— 投资主线 + 驱动假设 + **证伪条件** + **独立空头视角(Bear Case)** 对冲共识叙事 + 风险与跟踪指标。
7. **数据置信度总评 + 尽调追问清单** —— 报告头部标"能信几分"，末尾给"下一步该核实什么、该追问哪几个问题"。

## 怎么用

直接说，例如：
- "分析一下 AI 产业链"
- "深入挖掘高多层 PCB 领域"
- "帮我看看储能行业的投资机会"

Skill 会按 6 阶段工作流推进（先回显行业边界 → 联网取数 → 逐模块产出 → 生成 HTML 报告），默认 **A 股为主视角**、**联网实时检索关键数据并标注来源**。

## 设计原则

- **先回显、再开干**：确认行业边界后才做全量分析，避免方向跑偏。
- **结构化产出，不给空表**：直接给打分、判断、结论。
- **联网取数、标注来源**：关键数字标来源与时间，预测值与历史实际值分开标注。
- **主观判断有数据兜底**：三高打分挂财务校验，背离强制处理。
- **投资合规**：做逻辑映射与跟踪框架，不做买卖建议；每份报告带风险/证伪与免责声明。

---

## 已知边界（诚实清单）

- 数据来自公开网络，**可能滞后或有误**；AI 不验证真伪，只验证多源一致。
- 容易跟随"共识叙事"——已用**强制独立 Bear Case 卡片 + 证伪信号**对冲，但仍需你判断空头论据是否够硬。
- 三高与估值打分是**结构化的主观判断**，非量化回测。
- 不含实时行情、券商内部预测、一手调研。

→ 把它当成"帮你快速搭好认知地图、列出该追问的问题"的工具，而**最终判断永远是你自己的**。
