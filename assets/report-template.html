术语一句话解释
Beta（贝塔）板块/大盘整体涨跌带来的收益，"水涨船高"，不分个股好坏。
Alpha（阿尔法）跑赢板块平均的超额收益，靠选对环节/个股赚的"真本事"。
卡位点一条赛道里决定"谁卡住利润、别人绕不过去"的具体控制点（工序/资质/数据/独家客户）。
卖铲子不赌终端谁赢，只供应上游核心部件的稳赚策略。
微笑曲线产业链利润"两端高、中间低"：研发/品牌高，组装/制造低。
三高本报告打分维度：高增长 / 高利润 / 高壁垒，各 1–5 分。
价值迁移（↑迁入/↓迁出）未来 1–3 年利润/价值往哪个环节流动。迁入＝还会变更好；迁出＝可能已见顶。
price in市场已把某利好/预期反映进现价，再兑现也难再涨。
预期差（正/负）你的判断 vs 市场已定价的差。市场预期偏低＝正预期差(机会)；偏高＝负预期差(易杀估值)。
赔率 × 胜率赔率＝能赚多少÷可能亏多少；胜率＝判断成真的概率。
Bear Case / 空头视角故意站"看跌"立场，论证这套逻辑可能错在哪。
证伪条件 / 信号出现什么具体可量化数据，就说明判断错了、该退出。
PE / PB / PS市盈率/市净率/市销率，都是"贵不贵"的尺子，越高越贵。
PE(TTM)用最近滚动 12 个月真实盈利算的市盈率。
PEGPE ÷ 盈利增速。<1 偏便宜、>2 偏贵；盈利下滑时变负值、失真。
历史估值分位当前估值在过去 5 年里的百分位。100%＝历史最贵。
毛利率 / 净利率(收入-成本)÷收入 / 净利润÷收入，越高越能赚。
ROIC / ROE投入资本回报率 / 净资产收益率，衡量"用钱赚钱"效率。
CR3 / CR5前 3 / 前 5 大公司市占率之和，越高越集中、越易出龙头。
渗透率 / S 曲线新产品被采用的比例；S 曲线"陡升段"＝放量最快阶段。
卡脖子 / 国产替代被海外垄断的进口依赖环节；国产替代＝国产厂商抢占份额。
