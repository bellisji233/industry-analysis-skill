# 标的梳理框架（逻辑映射，非投资推荐）

目的：把"高价值环节/细分方向"映射到具体上市公司，让用户知道**沿着哪条逻辑去看哪些标的、跟踪什么**。
合规底线：不给买卖建议、不给目标价；只做产业逻辑与定位映射，并提示风险与证伪。

## 一、按角色分类

### 1. 龙头 / 确定性标的
- 环节绝对龙头、CR3 内、份额稳定或提升
- 逻辑：行业 β + 份额提升 α，确定性强、弹性相对小
- 跟踪：市占率变化、产能利用率、新客户/新品放量

### 2. 弹性标的
- 体量中等、卡在高增长细分、业绩对景气/单一客户高度敏感
- 逻辑：高弹性，向上空间大但波动大
- 跟踪：单一变量（大客户订单、某产品价格、产能爬坡）

### 3. 隐形冠军
- 细分小环节的绝对龙头，常被忽视，高 ROE/高壁垒
- 逻辑：小而美、高壁垒、议价强
- 跟踪：下游需求扩散、份额天花板、出海

### 4.（可选）平台 / 一体化玩家
- 跨多环节布局，享受价值链整合红利
- 跟踪：一体化降本效果、各业务协同

## 二、每个标的必填字段
- **名称 / 代码**（A 股优先，标注港股/美股）
- **所处环节**（对应产业链图谱位置）
- **主营结构**（收入拆分，看纯度/相关度）
- **市占率 / 行业地位**
- **核心投资逻辑**（一句话：为什么属于高价值方向）
- **催化剂**（未来 6–12 月可能的正向变量）
- **估值锚**（PE/PB/PS 区间或对标，标注当前位置 高/中/低）
- **风险点**

## 三、映射表呈现
按"产业链环节 → 角色 → 标的"组织，让用户一眼看到：
某高价值环节里，谁是龙头、谁有弹性、谁是隐形冠军。

## 四、纯度提示
标的"主营纯度"很关键：
- 高纯度（主业=该环节）→ 直接受益，逻辑干净
- 低纯度（仅小部分业务相关）→ 弹性被稀释，需提示"占比小"
