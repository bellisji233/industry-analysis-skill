# 常见产业链结构模板

用作产业链拆解的起点骨架。实际分析按具体行业增删环节，并联网核验玩家与价值量。

## 通用三段式
- **上游**：资源/原材料 → 核心零部件 → 关键设备 → 技术IP
- **中游**：制造/加工 → 集成/封装 → 模组/系统
- **下游**：应用场景 → 渠道/集成商 → 终端客户/需求

---

## 1. 先进制造 / 硬科技（半导体、锂电、光伏、机器人）
- 上游：原材料（硅料/锂矿/稀土）、核心材料、设备（光刻/镀膜/激光）、EDA/IP
- 中游：晶圆制造/电芯制造/组件、封测/PACK、模组
- 下游：终端品牌、系统集成、应用（汽车/储能/消费电子）
- 价值高地：**设备 + 核心材料 + 龙头制造**；微笑曲线两端
- A 股看点：设备/材料国产化率，先进制程突破，扩产周期

## 2. 消费品（食品饮料、美妆、家电）
- 上游：原料/包材/代工厂（OEM/ODM）
- 中游：品牌商、配方/研发
- 下游：经销/线上线下渠道、消费者
- 价值高地：**品牌 + 渠道**（微笑曲线右端）
- 看点：渗透率、复购、提价能力、渠道变革（线上/会员店/兴趣电商）

## 3. 软件 / 互联网 / AI
- 上游：算力（芯片/云）、数据、基础大模型、开发框架
- 中游：平台/中间件、模型层、Agent/应用框架
- 下游：行业应用、SaaS、终端用户
- 价值高地：**算力 + 平台/模型 + 高粘性应用**
- 看点：用户/付费转化、留存、单位经济、Token 成本曲线、生态壁垒

## 4. 能源 / 新能源（储能、风电、氢能、电网）
- 上游：资源/原材料、核心部件（电芯/叶片/电解槽）、设备
- 中游：系统集成（PCS/EPC）、制造
- 下游：发电/电网/工商业/户用、运营
- 价值高地：**核心部件 + 系统集成龙头 + 运营资产**
- 看点：装机增速、度电成本、政策补贴/电价机制、出海

## 5. 医药 / 生物（创新药、器械、CXO）
- 上游：原料药/试剂、科研仪器、上游耗材
- 中游：研发（Biotech/Pharma）、CXO（CRO/CDMO）、器械制造
- 下游：医院/药店/患者、医保支付
- 价值高地：**创新药专利 + 高壁垒器械 + 一体化 CXO**
- 看点：管线/临床进度、医保谈判、集采影响、出海 BD

## 6. 周期 / 资源（有色、化工、钢铁）
- 上游：矿/资源
- 中游：冶炼/加工
- 下游：制造/基建/地产
- 价值高地：**稀缺资源端**（景气期）；出清期看格局优化的中游龙头
- 看点：供需缺口、库存周期、产能周期位置（务必标注周期位置）

---

## 拆链小贴士
- 先问"钱从哪来、到哪去"：跟着资金流和价值量走。
- 找"卡脖子/卡位"环节：技术/认证/资源稀缺处往往是价值高地。
- 区分"价值量大"与"利润高"：制造环节价值量大但利润薄，要分开看。
- 关注"环节合并/解耦"趋势：一体化（如锂电/光伏一体化）会改变价值分布。
